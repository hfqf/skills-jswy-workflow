| G | 差异 | `=E12-F12` | =目标单价-实际单价 |
| H | 线索金额 | `=B12*E12` | =线索数×目标单价 |
| I | 实际销售 | `=B12*F12` | =线索数×实际单价 |

### 关键参数 (从Sheet01引用)

|--------|------|------|
| B22 | 线索总数 | `=SUM(B6:B20)` |
| C22 | 有效线索 | `=SUM(C6:C20)` |

### 字段与公式

| 列 | 字段名 | 公式逻辑 | 数据来源 |
|----|--------|----------|----------|
| A | 来源渠道 | 手动输入 | 渠道名称 |
| B | 线索留资 | 手动输入 | 原始线索数 |
| C | 线索有效 | 手动输入 | 有效线索数 |

| E | 线索占比 | `=IF($B$22=0,0,B6/$B$22)` | =该渠道/全部 |
| F | 目标线索 | `=ROUND('01_3月经营台账'!$B$8*E6,0)` | =总数×占比 |

## 📊 Sheet 05: 销售个人报表

### 用途
销售个人绩效评分，运营评价

### 字段与公式

| 列 | 字段名 | 公式逻辑 | 数据来源 |
|----|--------|----------|----------|
| A | 排名 | `=Q12` | 引用P列排名 |
| B | 顾问 | 手动输入 | 销售顾问名 |
| C | 小组 | 手动输入 | 所属小组 |
| D-H | 各车型战败数 | 引用Sheet 03 | ='03_线索战败分解'!G6:L6 |
| I | 战败总数 | 引用Sheet 03 | ='03_线索战败分解'!R6 |
| J | 战败金额 | 引用Sheet 03 | ='03_线索战败分解'!P6 |
| K | 战败率 | `=IF($B$8=0,0,J12/$B$8)` | =战败/线索总数 |
| L | 战败排名 | `=_xlfn.RANK.EQ(J12,$J$12:$J$34,0)` | 降序排名 |
| M | 线索排名 | `=_xlfn.RANK.EQ(G12,$G$12:$G$34,0)` | 降序排名 |
| N | 预约到店率 | `=IF(G12=0,0,H12/G12)` | =到店/线索 |
| O | 到店排名 | `=_xlfn.RANK.EQ(N12,$N$12:$N$34,1)` | 升序排名（越低越好） |
| P | 综合得分 | `=IFERROR(((J12-MIN($J$12:$J$34))/(MAX($J$12:$J$34)-MIN($J$12:$J$34))*50+((G12-MIN($G$12:$G$34))/(MAX($G$12:$G$34)-MIN($G$12:$G$34))*30+((MAX($N$12:$N$34)-N12)/(MAX($N$12:$N$34)-MIN($N$12:$N$34))*20),0)` | 战败50%+线索30%+到店20% |
| Q | 综合排名 | `=_xlfn.RANK.EQ(P12,$P$12:$P$34,0)` | 降序 |
| R | 运营评价 | `=IF(J12<0,"重点关注战败超限",IF(Q12<=5,"重点培养积极跟进",IF(Q12<=12,"动态调整加强转化","资源倾斜提升效能")))` | ABCD评级 |

### 综合得分计算公式
```
综合得分 = 战败金额得分×50% + 线索数量得分×30% + 战败效果得分×20%

其中：
- 战败金额得分 = (个人战败金额-最小值)/(最大值-最小值) × 50
- 线索数量得分 = (个人线索数-最小值)/(最大值-最小值) × 30  
- 战败效果得分 = (最大值-个人到店率)/(最大值-最小值) × 20
```

### 运营评价规则
| 条件 | 评价 |
|------|------|
| 战败金额<0 | 重点关注战败超限 |
| 综合排名≤5 | 重点培养积极跟进 |
| 综合排名≤12 | 动态调整加强转化 |
| 综合排名>12 | 资源倾斜提升效能 |

### 数据关联
- 全部数据引用自Sheet 03
- 汇总指标：B5(顾问数), B6(线索总数), B7(战败总数), B8(有效顾问数)
