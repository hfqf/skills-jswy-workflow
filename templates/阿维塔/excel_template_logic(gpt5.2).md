## Sheet 01：01_3月经营总表

**用途**：3月核心经营指标、分车型毛利/费用/利润与经营利润汇总。

### 核心参数（输入/计算）

| 单元格 | 字段 | 公式/来源 |
|---|---|---|
| B5 | 3月销售目标（台） | 手动输入 |
| B6 | 其中库存车销售目标（台） | 手动输入 |
| B7 | 3月目标成交率 | 手动输入 |
| B8 | 3月线索目标（条） | `=ROUND(B5/B7,0)` |

### 分车型明细（Row 12-15）

| 列 | 字段 | 公式/来源 |
|---|---|---|
| A | 车型 | 手动输入（阿维塔06/07/11/12） |
| B | 3月销量目标(台) | 手动输入 |
| C | 库存车目标(台) | 行12-14：`=ROUND(IF($B$5=0,0,B12/$B$5*$B$6),0)`；行15：`=$B$6-SUM(C12:C14)` |
| D | 单车成交均价(元) | 手动输入 |
| E | 单车毛利(元) | 手动输入 |
| F | 单车变动费用(元) | 手动输入 |
| G | 单车贡献利润(元) | `=E12-F12` |
| H | 毛利总额(元) | `=B12*E12` |
| I | 变动费用总额(元) | `=B12*F12` |
| J | 贡献利润总额(元) | `=B12*G12` |
| K | 说明 | 手动输入 |

### 合计行（Row 16）

| 单元格 | 字段 | 公式 |
|---|---|---|
| B16 | 3月销量目标合计 | `=SUM(B12:B15)` |
| C16 | 库存车目标合计 | `=SUM(C12:C15)` |
| G16 | 单车贡献利润 | `=IF(B16=0,0,J16/B16)` |
| H16 | 毛利总额合计 | `=SUM(H12:H15)` |
| I16 | 变动费用总额合计 | `=SUM(I12:I15)` |
| J16 | 贡献利润总额合计 | `=SUM(J12:J15)` |

### 费用与利润汇总（Row 19-23）

| 单元格 | 字段 | 公式 |
|---|---|---|
| B19 | 销售收入（估算） | `=SUMPRODUCT(B12:B15,D12:D15)` |
| B20 | 车型贡献利润（毛利-变动费用） | `=J16` |
| B21 | 门店固定费用（来自明细表） | `=163.23*10000` |
| B22 | 预计经营利润（扣门店固定费用） | `=B20-B21` |
| B23 | 单车经营利润 | `=IF(B16=0,0,B22/B16)` |

## Sheet 02：02_线索来源分解

**用途**：渠道级线索目标分解 + 门店目标与费用明细。

### 一、渠道级3月目标（Row 6-21）

| 列 | 字段 | 公式/来源 |
|---|---|---|
| A | 渠道 | 手动输入 |
| B | 来源线索 | 手动输入 |
| C | 来源订单 | 手动输入 |
| D | 来源成交率 | 手动输入 |
| E | 线索占比 | `=IF($B$22=0,0,B6/$B$22)` |
| F | 3月线索目标 | `=ROUND('01_3月经营总表'!$B$8*E6,0)` |
| G | 3月订单目标 | `=ROUND('01_3月经营总表'!$B$5*IF($C$22=0,0,C6/$C$22),0)` |
| H | 目标成交率 | `=IF(F6=0,0,G6/F6)` |
| I | 建议CPL | 手动输入 |
| J | 预算(元) | `=F6*I6` |
| K | 负责人 | 手动输入 |

### 渠道汇总（Row 22）

| 单元格 | 字段 | 公式 |
|---|---|---|
| B22 | 来源线索合计 | `=SUM(B6:B21)` |
| C22 | 来源订单合计 | `=SUM(C6:C21)` |
| D22 | 汇总成交率 | `=IF(B22=0,0,C22/B22)` |
| E22 | 线索占比合计 | `=SUM(E6:E21)` |
| F22 | 线索目标合计 | `=SUM(F6:F21)` |
| G22 | 订单目标合计 | `=SUM(G6:G21)` |
| H22 | 目标成交率合计 | `=IF(F22=0,0,G22/F22)` |
| I22 | 平均CPL | `=IF(F22=0,0,J22/F22)` |
| J22 | 预算合计 | `=SUM(J6:J21)` |

### 二、门店目标与费用明细（Row 26-31）

| 列 | 字段 | 公式/来源 |
|---|---|---|
| A | 门店 | 手动输入 |
| B | 3月销量(台) | 手动输入 |
| C | 3月销量目标(台) | 手动输入 |
| D | 库存车目标(台) | `=ROUND(C26*0.4,0)` |
| E | 3月线索目标(条) | `=ROUND('01_3月经营总表'!$B$8*IF('01_3月经营总表'!$B$5=0,0,C26/'01_3月经营总表'!$B$5),0)` |
| F | 单店费用(万元) | 手动输入 |
| G | 单店费用(元) | `=F26*10000` |
| H | 单车固定费用(元/台) | `=IF(C26=0,0,G26/C26)` |
| I-T | 各渠道线索分配 | `=ROUND(E26*$F$6/$F$22,0)` 到 `=ROUND(E26*$F$17/$F$22,0)` |
| U | 其他渠道 | `=E26-SUM(I26:T26)` |
| V | 负责人 | 手动输入 |

### 门店汇总（Row 32）

| 单元格 | 公式 |
|---|---|
| B32 | `=SUM(B26:B31)` |
| C32 | `=SUM(C26:C31)` |
| D32 | `=SUM(D26:D31)` |
| E32 | `=SUM(E26:E31)` |
| F32 | `=SUM(F26:F31)` |
| G32 | `=SUM(G26:G31)` |
| H32 | `=SUM(H26:H31)` |
| I32-T32 | 对应列求和 |
| U32 | `=SUM(U26:U31)` |

## Sheet 05：05_销售分析排名

**用途**：销售人员经营分析与综合排名。

### 一、人员经营概览（Row 5-8）

| 单元格 | 字段 | 公式 |
|---|---|---|
| B5 | 销售顾问人数 | `=COUNT(B12:B34)` |
| B6 | 总销量目标(台) | `=SUM(G12:G34)` |
| B7 | 总线索目标(条) | `=SUM(H12:H34)` |
| B8 | 总预计经营利润(元) | `=SUM(J12:J34)` |
| E5 | 人均销量目标(台) | `=IF(B5=0,0,B6/B5)` |
| E6 | 人均线索目标(条) | `=IF(B5=0,0,B7/B5)` |
| E7 | 人均预计经营利润(元) | `=IF(B5=0,0,B8/B5)` |
| E8 | 利润为正人数 | `=COUNTIF(J12:J34,">0")` |

### 二、销售人员分析与排名（Row 12-34）

| 列 | 字段 | 公式/来源 |
|---|---|---|
| A | 排名 | `=Q12` |
| B | 门店 | `='03_门店顾问分解'!A6` |
| C | 销售顾问 | `='03_门店顾问分解'!B6` |
| D-F | 3月07/06/12目标 | `='03_门店顾问分解'!G6:H6:I6` |
| G | 3月合计目标 | `='03_门店顾问分解'!J6` |
| H | 个人线索目标 | `='03_门店顾问分解'!K6` |
| I | 个人库存目标 | `='03_门店顾问分解'!L6` |
| J | 预计经营利润(元) | `='03_门店顾问分解'!R6` |
| K | 利润贡献占比 | `=IF($B$8=0,0,J12/$B$8)` |
| L | 利润排名 | `=_xlfn.RANK.EQ(J12,$J$12:$J$34,0)` |
| M | 销量排名 | `=_xlfn.RANK.EQ(G12,$G$12:$G$34,0)` |
| N | 线索效率(线索/台) | `=IF(G12=0,0,H12/G12)` |
| O | 效率排名 | `=_xlfn.RANK.EQ(N12,$N$12:$N$34,1)` |
| P | 综合得分 | `=IFERROR(((J12-MIN($J$12:$J$34))/(MAX($J$12:$J$34)-MIN($J$12:$J$34))*50)+((G12-MIN($G$12:$G$34))/(MAX($G$12:$G$34)-MIN($G$12:$G$34))*30)+((MAX($N$12:$N$34)-N12)/(MAX($N$12:$N$34)-MIN($N$12:$N$34))*20),0)` |
| Q | 综合排名 | `=_xlfn.RANK.EQ(P12,$P$12:$P$34,0)` |
| R | 经营建议 | `=IF(J12<0,"优先帮扶：优化费用与线索质量",IF(Q12<=5,"重点冲刺：保持高产并带 教",IF(Q12<=12,"稳态提升：强化库存去化","跟进改善：提升转化效率")))` |

### 三、汇总与榜单（Row 35/41）

| 单元格 | 公式 |
|---|---|
| D35-K35 | `=SUM(D12:D34)` ~ `=SUM(K12:K34)` |
| N35 | `=AVERAGE(N12:N34)` |
| P35 | `=AVERAGE(P12:P34)` |
