## Sheet 1: 01_3月经营总表

**用途**：3月核心经营指标汇总，展示各车型销量目标、毛利、费用、利润

### 字段说明

| 字段 | 示例值 | 数据来源/计算逻辑 |
|------|--------|-------------------|
| 3月销售目标(台) | 150 | 手动输入，默认引用3月门店合计销量 |
| 其中库存车销售目标(台) | 60 | 手动输入，默认按40%设置，可调整 |
| 3月目标成交率 | 0.0255 | 手动输入，沿用模板渠道成交口径 |
| 3月线索目标(条) | 5882 | `=ROUND(B5/B7,0)` 由销量目标反推：销量÷成交率 |

### 分车型数据表（Row 12-15）

| 字段 | 列 | 说明 |
|------|-----|------|
| 车型 | A | 阿维塔06/07/11/12 |
| 3月销量目标(台) | B | 手动输入 |
| 库存车目标(台) | C | 手动输入 |
| 单车成交均价(元) | D | 手动输入 |
| 单车毛利(元) | E | 手动输入，可手工调整 |
| 单车变动费用(元) | F | 手动输入 |
| 单车贡献利润(元) | G | `=单车毛利(E) - 单车变动费用(F)` |
| 毛利总额(元) | H | `=单车毛利(E) × 销量目标(B)` |
| 变动费用总额(元) | I | `=单车变动费用(F) × 销量目标(B)` |
| 贡献利润总额(元) | J | `=单车贡献利润(G) × 销量目标(B)` |
| 说明 | K | 车型销售策略备注 |

### 合计行（Row 16）

| 字段 | 公式 |
|------|------|
| 3月销量目标合计 | `=SUM(B12:B15)` |
| 库存车目标合计 | `=SUM(C12:C15)` |
| 单车贡献利润 | `=IF(B16=0, 0, J16/B16)` |
| 毛利总额合计 | `=SUM(H12:H15)` |
| 变动费用总额合计 | `=SUM(I12:I15)` |
| 贡献利润总额合计 | `=SUM(J12:J15)` |

### 费用与利润汇总（Row 19-23）

| 字段 | 计算逻辑 |
|------|----------|
| 销售收入（估算） | `=SUM(单车成交均价 × 3月销量目标)` |
| 车型贡献利润（毛利-变动费用） | `=J16` 从合计行引用 |
| 门店固定费用（来自明细表） | 从 Sheet2 门店费用汇总引用 |
| 预计经营利润（扣门店固定费用） | `=车型贡献利润 - 门店固定费用` |
| 单车经营利润 | `=预计经营利润 / 总销量` |

## Sheet 2: 02_线索来源分解

**用途**：渠道级线索目标分解 + 门店目标与费用明细

### 第一部分：渠道级3月目标（Row 7-16）

| 字段 | 列 | 说明 | 计算逻辑 |
|------|-----|------|----------|
| 渠道 | A | 各渠道名称 | 手动输入 |
| 来源线索 | B | 历史数据 | 手动输入 |
| 来源订单 | C | 历史数据 | 手动输入 |
| 来源成交率 | D | 历史成交率 | `=来源订单(C)/来源线索(B)` |
| 线索占比 | E | 该渠道线索占比 | `=B17/$B$22` （该渠道线索/总线索） |
| 3月线索目标 | F | 月度线索目标 | `=ROUND('01_3月经营总表'!$B$8 * E17, 0)` |
| 3月订单目标 | G | 月度订单目标 | `=ROUND('01_3月经营总表'!$B$5 * IF($C$22=0,0,C17/$C$22), 0)` |
| 目标成交率 | H | 目标成交率 | `=IF(F17=0, 0, G17/F17)` |
| 建议CPL | I | 单条线索成本 | 手动输入 |
| 预算(元) | J | 渠道预算 | `=F17 * I17` |

**关键引用**：
- `'01_3月经营总表'!$B$8` = 3月线索目标（5882）
- `'01_3月经营总表'!$B$5` = 3月销售目标（150）

### 第二部分：门店目标与费用明细（Row 26-32）

| 字段 | 列 | 说明 | 计算逻辑 |
|------|-----|------|----------|
| 门店 | A | 6个上海门店 | 手动输入 |
| 3月销量(台) | B | 当月实际销量 | 手动填写 |
| 3月销量目标(台) | C | 月度目标 | 手动输入 |
| 库存车目标(台) | D | 库存销售目标 | 手动输入 |
| 3月线索目标(条) | E | 线索目标 | 按门店销量占比分配 |
| 单店费用(万元) | F | 门店月费用 | 导入门店及AC明细表 |
| 单店费用(元) | G | | `=F*10000` |
| 单车固定费用(元/台) | H | | `=G/C` |
| 厂家渠道~其他渠道 | I-Q | 各渠道线索分配 | 按比例分配渠道线索到门店 |

## Sheet 3: 03_门店顾问分解

**用途**：销售顾问个人目标与利润分解

### 第一部分：销售顾问分解（Row 6-24）

| 字段 | 列 | 说明 | 计算逻辑 |
|------|-----|------|----------|
| 门店 | A | 所属门店 | 手动输入 |
| 销售顾问 | B | 顾问姓名 | 手动输入 |
| 3月07/06/12 | C-E | 各车型销量分配 | 手动分配 |
| 3月合计 | F | 总销量 | `=C+D+E` |
| 3月07/06/12目标 | G-I | 月度目标 | 手动输入（可调整） |
| 3月合计目标 | J | | `=G+H+I` |
| 个人线索目标 | K | | `=J * (总线索目标/总销量)` |
| 个人库存目标 | L | | `=J * (总库存目标/总销量)` |
| 单车毛利 | M | | 从 Sheet1 车型数据匹配 |
| 单车变动费用 | N | | 从 Sheet1 车型数据匹配 |
| 单车贡献利润 | O | | `=M - N` |
| 贡献利润总额 | P | | `=J * O` |
| 分摊门店费用 | Q | | `=门店总费用 / 门店顾问数` 或按销量比例 |
| 预计经营利润 | R | | `=P - Q` |
| 备注 | S | | 手动填写 |

**关键公式**：
- 单车毛利/变动费用：`=VLOOKUP(A6, '02_线索来源分解'!$A$26:$G$200, 7, 0)` 从门店明细匹配

### 第二部分：门店汇总校验（Row 27-33）

| 字段 | 说明 |
|------|------|
| 门店目标 | 从 Sheet1 引用 |
| 顾问汇总 | `=SUM(该门店所有顾问目标)` |
| 差异 | `=门店目标 - 顾问汇总` |
| 状态 | `=IF(差异=0, "一致", "不一致")` |
| 门店费用(元) | 从明细表引用 |
| 顾问分摊费用 | `=SUM(该门店所有顾问分摊费用)` |
| 差异 | `=门店费用 - 顾问分摊费用` |

## Sheet 5: 05_销售分析排名

**用途**：销售人员经营分析与综合排名

### 第一部分：人员经营概览（Row 6-9）

| 字段 | 说明 | 计算逻辑 |
|------|------|----------|
| 销售顾问人数 | 23 | `=COUNT('03_门店顾问分解'!B:B)` |
| 总销量目标(台) | 150 | `=SUM('03_门店顾问分解'!J:J)` |
| 总线索目标(条) | 5886 | `=SUM('03_门店顾问分解'!K:K)` |
| 总预计经营利润(元) | -627300 | `=SUM('03_门店顾问分解'!R:R)` |
| 人均销量目标 | | `=总销量/人数` |
| 人均线索目标 | | `=总线索/人数` |
| 人均预计经营利润 | | `=总利润/人数` |
| 利润为正人数 | 3 | `=COUNTIF(利润列, ">0")` |

### 第二部分：销售人员分析与排名（Row 12-34）

| 字段 | 列 | 说明 | 计算逻辑 |
|------|-----|------|----------|
| 排名 | A | 综合排名 | `=RANK.EQ(P12, $P$12:$P$34, 0)` |
| 门店 | B | | 从 Sheet3 引用 |
| 销售顾问 | C | | 从 Sheet3 引用 |
| 3月07/06/12目标 | D-F | | 从 Sheet3 引用 |
| 3月合计目标 | G | | 从 Sheet3 引用 |
| 个人线索目标 | H | | 从 Sheet3 引用 |
| 个人库存目标 | I | | 从 Sheet3 引用 |
| 预计经营利润(元) | J | | `='03_门店顾问分解'!R6` |
| 利润贡献占比 | K | | `=IF($B$8=0, 0, J12/$B$8)` |
| 利润排名 | L | | `=RANK.EQ(J12, $J$12:$J$34, 0)` |
| 销量排名 | M | | `=RANK.EQ(G12, $G$12:$G$34, 0)` |
| 线索效率(线索/台) | N | | `=IF(G12=0, 0, H12/G12)` |
| 效率排名 | O | | `=RANK.EQ(N12, $N$12:$N$34, 1)` （升序，效率越低越好） |
| 综合得分 | P | | 详见下方公式 |
| 综合排名 | Q | | `=RANK.EQ(P12, $P$12:$P$34, 0)` |
| 经营建议 | R | | 手动填写 |

### 综合得分公式

```
=IFERROR(
  ((J12-MIN($J$12:$J$34))/(MAX($J$12:$J$34)-MIN($J$12:$J$34))*50) +
  ((G12-MIN($G$12:$G$34))/(MAX($G$12:$G$34)-MIN($G$12:$G$34))*30) +
  ((MAX($N$12:$N$34)-N12)/(MAX($N$12:$N$34)-MIN($N$12:$N$34))*20),
  0
)
```

**权重分配**：
- 利润得分：50%
- 销量得分：30%
- 效率得分：20%（效率越高越好，故用 MAX-效率）

### 经营建议分类

| 综合得分 | 建议 |
|----------|------|
| ≥80 | 稳态提升：强化库存去化 |
| 70-80 | 重点冲刺：保持高产并带教 |
| <70 | 优先帮扶：优化费用与线索质量 |

## 关键计算公式汇总

| 场景|------|------ | 公式 |
|
| 线索目标反推 | `=ROUND(销量目标/成交率, 0)` |
| 单车贡献利润 | `=单车毛利 - 单车变动费用` |
| 贡献利润总额 | `=单车贡献利润 × 销量目标` |
| 预计经营利润 | `=贡献利润总额 - 门店固定费用` |
| 门店单车固定费用 | `=门店总费用 / 门店销量目标` |
| 顾问分摊门店费用 | `=门店总费用 / 门店顾问数` |
| 顾问预计利润 | `=个人贡献利润 - 分摊门店费用` |
| 周度目标分解 | `=月度目标 / 4` |
| 达成率 | `=IF(目标=0, 0, 实际/目标)` |
| 利润排名 | `=RANK.EQ(利润, 区域, 0)` |
| 综合得分 | `=利润标准化×50% + 销量标准化×30% + 效率标准化×20%` |
